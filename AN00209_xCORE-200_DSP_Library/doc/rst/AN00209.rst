Introduction
------------

The XMOS xCORE-200 DSP Elements Library provides foundation Digital
Signal Processing functions.

The library is split into the following modules :

   * Basic Math - xmos_dsp_math.h
   * Filtering - xmos_dsp_filters.h
   * Vectors - xmos_dsp_vector.h
   * Statistics - xmos_dsp_statistics.h
   * Adaptive Filters - xmos_dsp_adaptive.h
   * Matrix Math - xmos_dsp_matrix.h

The library supports variable Q formats from Q16 to Q31, using the
macros in the following header file : xmos_dsp_qformat.h.

In order to use any of these modules and the Q formats it is
only necessary to include the following header file :

#include "xmos_dsp_elements.h"

Using The Test Applications
---------------------------

The test functions support all of the functions in the library :

   * Basic Math - math.c
   * Filtering - filters.c
   * Vectors - vector.c
   * Statistics - statistics.c
   * Adaptive Filters - adaptive.c
   * Matrix Math - matrix.c

All of the test functions are written using Q24 format.
